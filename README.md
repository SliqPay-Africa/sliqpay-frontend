## 🌍 What is SliqPay?

SliqPay is a universal fintech platform built for Africa that lets users pay everyday bills — airtime, data, electricity, and cable TV — using both traditional fiat balances and cryptocurrency (AVAX on Avalanche). 

Africa has over 600 million mobile money users but fragmented infrastructure. SliqPay bridges Web2 bill payment rails (VTPass) with Web3 wallets (Avalanche), giving users a single interface to pay anything, with anything.

**Live Demo:** [sliqpay-frontend.vercel.app](https://sliqpay-frontend.vercel.app)

## ✨ Core Features

### 💳 Dual Payment Rails
Users can pay bills using either their fiat wallet balance or AVAX from an external Web3 wallet (MetaMask, etc.). Both flows converge on the same success experience.

### 🔗 Crypto → Real World Bills
When a user pays with AVAX:
1. MetaMask prompts to send AVAX to the **TreasuryVault** smart contract on Avalanche Fuji
2. The backend listens, verifies the on-chain transaction via Fuji RPC
3. VTPass API is called to fulfill the bill payment in the real world
4. The transaction is recorded with full audit trail in PostgreSQL

### 🪪 SliqID — On-Chain Identity
Every SliqPay user gets an auto-generated **SliqID** (e.g. `@User_4415`) which is:
- Generated by the backend at signup
- Registered on-chain via the `SliqIDRegistry` smart contract
- Used as a universal payment handle across the ecosystem

### 🔐 Embedded + External Wallet Support
- **Magic SDK** for seamless embedded wallets (no seed phrase for new users)
- **wagmi + WalletConnect** for external wallets (MetaMask, Phantom, etc.)
- Users can choose their preferred Web3 experience

**Chain:** Avalanche Fuji Testnet (Chain ID: 43113)
**Explorer:** [testnet.snowtrace.io](https://testnet.snowtrace.io)

### TreasuryVault
Receives AVAX payments from users. The backend verifies deposits before triggering any bill payment, preventing double-spends and unauthorized calls.

### SliqIDRegistry
Maps every SliqID handle to a wallet address on-chain. This creates a decentralized identity layer that is interoperable with any dApp in the Avalanche ecosystem.

### MockFxOracle
Provides simulated AVAX/NGN exchange rates for the demo. In production, this would be replaced by a Chainlink price feed.

## 🚀 Running Locally

### Prerequisites
- Node.js 18+
- MetaMask browser extension
- Testnet AVAX (from [core.app/tools/testnet-faucet](https://core.app/tools/testnet-faucet))

### Frontend
```bash
git clone https://github.com/SliqPay-Africa/sliqpay-frontend
cd sliqpay-frontend
npm install
cp .env.example .env.local
# Fill in your env vars (see below)
npm run dev
```

### Backend
```bash
git clone https://github.com/SliqPay-Africa/sliqpay-backend
cd sliqpay-backend
npm install
# Set DATABASE_URL, JWT_SECRET, VTPASS keys in .env
npx prisma db push
npm run dev
```

### Smart Contracts
```bash
git clone https://github.com/SliqPay-Africa/sliqpay-smartcontracts
cd sliqpay-smartcontracts
npm install
# Set PRIVATE_KEY and RPC_URL in .env
npx hardhat compile
npx hardhat run scripts/deploy.ts --network fuji
```

## 💳 Demo Flow

1. **Sign up** at [sliqpay-frontend.vercel.app/auth/signup](https://sliqpay-frontend.vercel.app/auth/signup)
2. A **SliqID** is auto-generated for you (e.g. `@User_4415`)
3. Connect your **MetaMask** wallet (switch to Avalanche Fuji testnet)
4. Navigate to **Dashboard → Utilities → Buy Airtime**
5. Select network, enter phone number, amount
6. Choose **Crypto (AVAX)** as payment method
7. Enter your transaction PIN
8. **MetaMask** prompts — approve the AVAX transfer to TreasuryVault
9. Backend verifies on-chain → VTPass fulfills airtime purchase
10. ✅ Success!
